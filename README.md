## Tom-Troc

## Pour utiliser ce projet : 

- Commencer par cloner le projet. 
- installez le projet chez vous, dans un dossier exécuté par un serveur local (type MAMP, WAMP, LAMP, etc...)
- Une fois installé chez vous, créez un base de données vide appelée : "tom-troc"
- Importez le fichier "tom-troc.sql" dans votre base de données.

## Lancez le projet ! 

Le fichier config.php contient notamment les informations de connextion à la base de données. 

Lancer le projet en pointant votre navigateur vers index.php

## Problèmes courants :

Ce projet a été réalisé avec PHP 8.3.28 Bien que d'autres versions de PHP puissent fonctionner, il n'est pas garanti que le projet fonctionne avec des versions antérieures.
